Afin de lancer le jeu, il faut :
- Avoir téléchargé grâce à pip toutes les dépendences décrites dans Requirements.txt qui ne sont pas natives à python 3.13
- Exéctuer main.py

Pour quitter le jeu, on peut : 
- Appuyer sur le bouton "QUIT" sur le menu principal
- Appuyer sur la touche Echap

Il est possible de changer les contrôles en modifiant les valeurs de la colonne "key" dans keybinds.csv
